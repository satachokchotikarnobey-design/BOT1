• ตั้งค่า DPI ในเกมให้เป็น 2000
• ปรับ Sensitivity ให้เหมาะสม
• ปิด Anti-virus ชั่วคราว (ถ้าใช้ GG)

📌 วิธีใช้ Magic Bullet

  1. กด ตั้งค่า DPI 2000 แล้วตั้งค่าในเกมตามที่แนะนำ
  2. กด ยิงตัวขึ้นหัว → จะได้สคริปต์ไปใช้ใน GG
  3. ติดตั้ง GG จากปุ่มข้างล่าง (ถ้ายังไม่มี)
  4. เปิด GG → เลือก FreeFire → Execute Script → วางสคริปต์ → Run
  5. กลับเข้าเกม ยิงที่ไหนก็เฮดชอต
